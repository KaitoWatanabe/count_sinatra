count-sinatra
====

## Requirement
* Ruby 2.0 ~ 

## Install
```
git clone git@github.com:KaitoWatanabe/count_sinatra.git
cd count_sinatra
bundle install
bundle exec rake db:migrate
```

## Usage

```
ruby app.rb
```

Open `localhost:4567/count` in your web-browser
